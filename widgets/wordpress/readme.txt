Automate customer support with ChatbotGPT, the best custom ChatGPT Agent provider. Free WordPress Plugin.

== Description ==

ChatbotGPT is a no-code platform for building ChatGPT bots trained on custom data. To provide this functionality, the plugin requires a remote JavaScript file hosted on an external server for seamless integration. Rest assured, the remote file is essential for the proper operation of ChatbotGPT and has been carefully selected for safety and reliability.

**Important Note:** The remote file hosted at `https://cdn.jsdelivr.net/npm/@databerry/chat-bubble@latest` is used to embed ChatbotGPT into your WordPress website. It is safe and integral to the functionality of the plugin, providing a smooth and secure chat experience for your users.

[youtube https://www.youtube.com/watch?v=Pa-cyPJzt5k]

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/chaindesk` directory, or install the plugin through the WordPress plugins screen directly.
2. Use the Settings->ChatbotGPT screen and click on "Link with ChatbotGPT"
